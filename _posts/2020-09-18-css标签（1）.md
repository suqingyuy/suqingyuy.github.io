##css 样式表

- css样式表和HTML有三种关联方式

   - 1: 通过link标签方式链接 （外联）
   - 2：在style标签中书写css（内联）
   - 3：style属性值是css代码（内嵌）
   
 
- 书写规范

- 选择器：要为页面中的那一块设置样式
>- 基础选择器:
 - *:通配符选择器
  ``` 
     *{
     }  
  ```

 - .类名:类选择器，给标签添加一个class标签  例：(class="a")
 ```
    .a{
     }
```
 - #id:给标签添加一个id属性 例：（id="a")  '#a{}'
